# Journal

Time-stamped notes. Meeting notes, daily logs, session journals, decisions in flight, observations to revisit.

## What goes here vs. elsewhere

- **Journal** = time-stamped, mostly chronological (e.g. `2026-04-23 — Wiki rebuild session.md`)
- **Decision** = formal decision record (`decisions/` template; can be cross-linked from journal)
- **Question** = filed answer (use `questions/`)
- **Meeting** = use `meeting.md` template (lives here)

## Folder convention

Flat by default. Filename starts with date: `YYYY-MM-DD — <topic>.md`. Group by year/month if you accumulate more than ~50 entries.
